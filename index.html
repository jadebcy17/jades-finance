GitHub Pages
RECOMMANDE — 100% gratuit
Heberge ton app en ligne → toutes les APIs fonctionnent automatiquement. 5 minutes.
  1. Va sur github.com → connecte-toi (jadebcy17)
  2. Cree un nouveau repo: jades-finance
  3. Upload ce fichier HTML → commit
  4. Settings → Pages → Branch: main → Save
  5. Ton app sera sur jadebcy17.github.io/jades-finance
